# XMacro Bridge 版本说明

## 1.1.0（2026-07-13）

首个功能汇总版本，将 1.0 系列的实机修复、嵌套宏编辑、宏库排序和整套 UI/UX 精修整理为统一发布包：

- 宏库 XMouse 与雷蛇 XML 列表支持绿色插入线拖拽排序，自定义顺序持久化保存，并随重命名和分组移动同步更新。
- 工作区文件、拖放和宏文本导入改为追加模式，新导入宏自动选中，已有宏不再被清空。
- 独立雷云 XML 保留真实根 `Guid`，正确处理一基 `MPIndex`；时间线支持为缺失嵌套引用手动绑定目标宏或插入新的嵌套宏事件。
- 键盘插入和事件编辑统一使用直接按键捕获及可读名称，修复中文输入法包装事件导致的 `VK 229` 误识别。
- 固定使用 Razer 暗色界面，完善无宏、空时间线、诊断筛选、宏库空列表和忙碌状态；增加危险操作层级、统一暗色确认框及首错聚焦。
- 增加 `Ctrl+O`、`Ctrl+F`、`Ctrl+Z`、`Ctrl+Y` 与时间线搜索快捷键；设置页显示应用版本和第三方许可入口。
- 目标格式、普通下拉框和嵌套目标使用局部绑定，避免对象类型名、空白选中项或全局样式回归。
- 继续保留输入只读、原子导出、100,000 事件资源上限、时间线虚拟化和 PerMonitorV2。

## 1.0.9（2026-07-13）

- XMouse 与雷蛇 XML 宏库列表支持鼠标拖拽排序，并显示绿色插入位置。
- 自定义顺序写入宏库 `.xmacrobridge/metadata.json`，刷新或重启后继续保留；重命名、移动分组时排序元数据同步更新。
- 搜索、回收站和“按时间排序”状态下禁用手动拖拽，避免排序规则冲突。

- 导入状态显示本次新增数量和当前工作区宏总数。
