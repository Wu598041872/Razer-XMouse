# XMacro Bridge 版本说明

## 0.1.0-preview.3（2026-07-12）

第二轮可用性修复：

- 识别雷云 XML 与 `.synapse4` 中复用的 `Type=6/Number/LoopEvent` 有限循环，不再误报“缺少 Delay”。
- 将有限循环和嵌套循环安全展开为 XMBC 与雷云 XML 都能表达的线性事件，并重新生成连续序号。
- 对未配对、次数不一致、非法次数、嵌套深度和展开事件数超限提供阻断诊断；超大次数空循环不会造成无意义迭代。
- 修复雷云独立 XML 的键盘扫描码、左右鼠标按钮 `1/2` 编码及 Version 4 根元数据。
- 新增 XML、`.synapse4`、XMBC 输出、雷云往返、嵌套和损坏循环回归。

## 0.1.0-preview.2（2026-07-12）

紧急可用性修复：

- 支持雷云 XML 与 `.synapse4` 中 `Type=6`、`Delay=<毫秒>` 的固定延时事件。
- 修复这类延时被标记为未知事件并阻止导出的问题。
- 同一未知事件不再同时显示格式错误和重复的通用 `UNKNOWN_EVENT` 错误。
- 增加真实工作区导入及雷云/XMBC 双向导出回归。

## 0.1.0-preview.1（2026-07-12）

首个可用预览版，提供：

- 雷云 XML、`.synapse4`、XMBC XML/`.xmbcp` 和宏文本导入。
- 雷云独立宏 XML 与 XMBC 宏文本导出。
- 嵌套宏展开、状态平衡检查和不兼容事件诊断。
- 时间线延时与键鼠事件编辑、复制、删除、移动、搜索、撤销和重做。
- 浅色、深色、高对比度、PerMonitorV2 和键盘可访问性支持。
- 输入文件只读、目标原子替换、取消恢复和异常路径保护。

已知限制：

- 不生成 `.synapse4` 整包。
- 雷云扩展鼠标按钮及部分扩展事件尚未完成实机编码确认。
- XMBC 应用规则、层级和设备配置不在本版本转换范围内。
- 这是预览版本，重要宏必须保留原始备份并先做安全试运行。
