# 产品需求规格

## 1. 产品目标

XMacro Bridge 是一款 Windows 离线桌面工具，用于在 Razer Synapse 雷云 4 与 X-Mouse Button Control（XMBC）之间转换宏序列。产品优先保证行为可解释、输入文件安全和不兼容项可见。

## 2. 目标用户与核心流程

目标用户是需要迁移或调整鼠标宏、但不希望手工重录事件的 Windows 用户。

核心流程：导入文件/文件夹或直接输入 XMBC 宏文本 → 选择宏 → 展开依赖 → 查看时间线与诊断 → 编辑 → 选择目标格式 → 安全导出。

## 3. 第一版范围

### 输入

- 雷云独立宏 XML。
- 雷云 4 `.synapse4` JSON 容器及其 Base64 宏载荷。
- XMBC 的 `XMBCSettings.xml` 和 `.xmbcp`。
- XMBC 模拟按键宏文本。
- 单文件、多文件、文件夹、命令行路径和文本粘贴。

### 处理

- 键盘按下、释放和点击。
- 鼠标按下、释放和点击。
- 毫秒延时。
- 嵌套宏递归展开。
- 对无法自动匹配的嵌套事件，可从当前工作区已导入的其他宏中手动选择并绑定目标，也可直接插入新的嵌套宏事件。
- 时间线编辑、撤销、重做、搜索和延时缩放。
- 未知事件、缺失引用、循环引用及按键状态错误诊断。

### 输出

- 雷云独立宏 XML，不包含嵌套引用。
- XMBC 模拟按键宏文本，可复制或保存为 `.txt`。

### 宏库

- 顶部导航提供独立“宏库”页面，以用户选择的本地目录保存 XMouse `.txt` 与雷蛇 `.xml`。
- 根目录文件属于“未分组”，一级真实文件夹属于同名分组；其他附件保持原样且不参与宏列表。
- 两种格式分栏展示，支持名称/分组搜索、收藏、最近使用、复制入库、显式更新和库内回收站。
- 路径切换不移动旧资料；独立迁移流程复制全部内容、校验 SHA-256 后才切换。

## 4. 非目标

- 不生成新的 `.synapse4` 整包。
- 不修改雷云或 XMBC 的在线配置、数据库或运行进程。
- 不转换 DPI、灯光、轮询率、完整设备配置、XMBC 应用规则、层级、和弦或全局热键。
- 不提供云同步、账号系统或宏分享平台。

## 5. 成功与验收条件

- 现有已知样本能够稳定识别并产生结构化结果。
- 基础键鼠宏双向转换后事件顺序和总时长符合转换规则。
- 嵌套宏输出中不存在调用事件；缺失或循环引用时禁止无提示导出。
- 输入文件不被修改；中断导出不产生损坏的目标文件。
- 用户无需阅读技术文档即可完成一次转换。
- 产品可在 Windows 10/11 x64 解压运行，不需要管理员权限或网络。
- 用户可以从现有目录直接建立宏库，不必重排文件，并能从宏库一键打开到转换工作区。
