# 宏格式规范

> 本文记录已由真实样本确认的结构。推测字段必须标注“待验证”，不得作为稳定导出契约。

## 1. 雷云独立宏 XML

根元素为 `<Macro>`，常见子元素为 `<Name>` 和 `<MacroEvents>`。事件结构：

- `Type=0`：延时，`Number` 使用秒制小数。
- `Type=1`：键盘事件，`KeyEvent/Makecode` 为 Windows 键码，`State=0` 表示按下，`State=1` 表示释放。
- `Type=2`：鼠标事件，`MouseEvent/MouseButton` 表示按钮，`State` 表示按下或释放。
- `Type=7`：嵌套宏引用，已知字段包括 `MPIndex` 与 `guid`。
- `Type=actionBar`：编辑器元数据，不作为执行事件输出。

`Id`、`flag`、`selected`、`isPairing` 等字段需保留兼容形态，但不参与统一执行语义。

当前匿名化黄金样本确认 `MouseButton=0` 为左键、`MouseButton=1` 为右键。其他雷云鼠标按钮代码在获得可靠样本前不得猜测映射。

## 2. `.synapse4`

文件为 UTF-8 JSON，顶层已知字段包括 `productId`、`deviceName`、`category`、`profiles` 和 `macros`。`profiles` 与 `macros` 条目包含：

- `name`
- `payload`：Base64 编码的 UTF-8 JSON。
- `hash`：校验字段，算法尚未确认。

第一版只读取整包，不生成整包。宏载荷中包含 `guid`、`macroEvents` 和 `macroPlaybackOption`。

只读导入器对外层 JSON 和内层 Base64 JSON 分别设置大小与深度限制。单个损坏宏产生条目级诊断，不阻止同一整包内其他有效宏导入。`hash` 当前只记录为未验证信息，不作为拒绝读取依据。

## 3. XMBC 配置文件

`XMBCSettings.xml` 与 `.xmbcp` 均为 XML 配置。宏通常位于各应用、层级和按钮元素中：

- `action=28` 表示模拟按键序列。
- `keys` 保存宏文本。
- `keyaction` 表示执行方式。
- `keyrepeat` 表示重复间隔。
- `active`、`blockmouse`、`randomisedelay` 为附加行为。

导入时只提取 `action=28`，名称按应用、层级、按钮和 `desc` 组合生成。第一版不导出完整 XMBC 配置。

配置导入器同时识别默认配置、`Application` 配置、附加 `Layer1`–`Layer9` 和 `chords` 内的宏。XMBC XML 中直接位于配置下的按钮属于第 1 层，`Layer1` 对应用户界面的第 2 层，以此类推。来源的 `keyaction`、`keyrepeat`、`active`、`blockmouse` 和 `randomisedelay` 作为元数据保留，不擅自换算播放模式。

## 4. XMBC 宏文本

用户提供的 XMBC 软件内说明已确认以下类别：

- 辅助键：`CTRL/RCTRL`、`ALT/RALT`、`SHIFT/RSHIFT`、`LWIN/RWIN`、`APPS`；可组合，但只应用于下一个普通或扩展按键。
- 扩展键、方向键、`F1`–`F24`、音量、多媒体、浏览器和数字键盘标记。
- 鼠标左/右/中键、`MB4/XMB1`、`MB5/XMB2` 及对应 `D/U` 按下释放后缀。
- 滚轮与倾斜：`MWUP/MWDN/TILTL/TILTR`。
- 延时与按住：`WAIT/WAITMS`、随机 `WAITMS:x-y`、`HOLD/HOLDMS`。
- 原始键码：`VKC/EXT/SC/SCE`。
- 移动、剪贴板、程序、动作、Windows 消息、方案切换和触发条件标记。
- `PRESS/RELEASE` 会持续影响后续一串键，直到模式改变；所有显式按下必须在同一序列释放。

当前实现支持普通字母/数字/空格、辅助键组合、扩展/方向/功能/音量/多媒体/浏览器/数字键盘、`VKC/EXT/SC/SCE`、已确认鼠标标记、固定/随机延时、`HOLD/HOLDMS` 和多键 `PRESS/RELEASE`。移动、剪贴板、程序、动作、消息、方案与触发条件作为 XMBC 专属命令完整保留，可原样导回 XMBC；转换到雷云时产生阻断错误。

## 5. 待验证事项

- `.synapse4` 的 `hash` 算法及版本兼容性。
- 雷云其他事件类型、滚轮、鼠标移动和扩展按钮编码。
- XMBC 普通标点转义、扫描码、随机延时及不同键盘布局行为。
