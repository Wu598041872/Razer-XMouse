# 用户使用说明

## 1. 启动

解压绿色版 ZIP 后双击 `XMacroBridge.App.exe`。软件不需要安装、管理员权限或网络连接。请保持 ZIP 内文件的相对位置，不要只单独复制 EXE。

## 2. 导入宏

可使用“选择文件”“选择文件夹”，也可以把文件或文件夹拖入窗口。支持：

- Razer Synapse 雷云独立宏 XML。
- 雷云 4 `.synapse4` 包，只读提取其中宏。
- XMBC 的 XML、`.xmbcp` 和模拟按键宏文本。

导入不会修改、移动或重命名原文件。重新导入会清空当前工作区尚未导出的编辑，请先保存需要的结果。

## 3. 查看与编辑

在左侧选择宏后，主区域显示事件时间线和兼容性报告。可执行：

- 修改单个固定延时，或按百分比缩放全部固定/随机延时。
- 插入键盘 VK、扩展键、鼠标按钮及独立的按下/释放事件。
- 复制、删除、上移或下移事件。
- 按序号、类型或详情搜索，并循环定位上一个/下一个结果。
- 撤销或重做最近 20 步编辑。

编辑器不会自动补充按键释放。若宏结束时仍有按键或鼠标按钮处于按下状态，兼容性报告会显示错误并禁止导出。

## 4. 导出

选择目标格式：

- “雷云 4 宏 XML”：生成可单独导入雷云的 XML。
- “X-Mouse 宏文本”：生成可复制到 XMBC“模拟按键”的文本文件。

点击“安全导出”并选择新文件名。软件默认不覆盖原输入；覆盖已有输出时由 Windows 保存对话框再次确认。输出先写入同目录临时文件，完成后再替换目标。

## 5. 诊断说明

- 错误：会阻止导出，必须修复或改选目标格式。
- 警告：转换可以继续，但目标行为可能降级或需要确认。
- 信息：说明规范化、取消或其他非阻断状态。

未知事件不会被静默删除。如果目标格式无法安全表达某个事件，软件会保留可见诊断并停止导出。

## 6. 安全建议

- 始终保留雷云和 XMBC 原始宏备份。
- 首次转换后先在无风险应用中测试，不要直接用于不可逆操作。
- 不要运行来源不可信、包含程序启动或系统消息命令的宏。
- 软件日志不记录完整宏内容，但宏文件本身可能包含隐私或操作策略，请谨慎分享。

## 7. 卸载

退出软件后删除解压目录即可。软件不会安装系统服务或写入雷云/XMBC 配置。若已注册开发环境的日报定时任务，它不属于绿色发布包。
