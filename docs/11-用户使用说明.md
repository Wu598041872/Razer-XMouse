# 用户使用说明

## 1. 启动

解压绿色版 ZIP 后双击 `XMacroBridge.App.exe`。软件不需要安装、管理员权限或网络连接。请保持 ZIP 内文件的相对位置，不要只单独复制 EXE。

## 2. 导入宏

可使用“选择文件”“选择文件夹”，也可以把文件或文件夹拖入窗口。若手头已经有 XMBC 模拟按键序列，点击“输入宏文本”，直接粘贴或编辑内容后选择“导入到工作区”，无需先保存 `.txt` 文件。支持：

- Razer Synapse 雷云独立宏 XML。
- 雷云 4 `.synapse4` 包，只读提取其中宏。
- XMBC 的 XML、`.xmbcp` 和模拟按键宏文本。

导入不会修改、移动或重命名原文件。直接输入的文本只在本机内存中解析，不会生成临时输入文件。后续选择、拖入文件或输入宏文本会追加到当前工作区，并自动选中新加入的宏，不会清除已有宏。

雷云 4 的有限循环会在导入时按次数展开为线性事件，因为 XMBC 没有等价循环标记。循环配对或次数异常时会显示错误并阻止导出。

软件按 Version 4 原生规则读取和导出 Windows VK 及 `0/1` 左右键，同时兼容缺少 Version 的旧式扫描码与 `1/2` 鼠标输入，无需手工修改原文件。

## 3. 查看与编辑

在左侧选择宏后，主区域显示事件时间线和兼容性报告。可执行：

- 在左侧宏列表右键目标宏并选择“重命名…”。新名称会同时用于安全导出的建议文件名和雷云 XML 内的 `<Name>`，因此导入雷云 4 后也会显示新名称。
- 在左侧宏列表右键选择“修改宏文本…”，可直接编辑当前 XMBC 序列并应用；宏名称保持不变，时间线和诊断会重新生成，可使用撤销恢复。
- 时间线按雷云宏编辑器样式显示为“图标 → 按下/松开箭头 → 事件名称”；延时显示为 `0.330s` 等三位秒数，VK、扫描码和毫秒值可在工具提示中查看。
- 每行带有事件序号；连续点击不同事件行即可直接累积多选，无需按住 Ctrl。鼠标停留时会出现绿色边框、选择框、复制、删除和拖动手柄；选择框可单独加入或取消某一项。选中多个事件后，可从任一已选行或右侧手柄拖动，工具栏与行内按钮也会对整个选择执行复制、删除或成组移动。
- 拖动时鼠标旁会显示“移动 N 个操作”，目标位置出现绿色提示线：指向某行上半部分表示插入该行之前，指向下半部分表示插入该行之后。
- 修改单个固定延时，或按百分比缩放全部固定/随机延时。
- 插入或编辑键盘事件时点击按键输入框并直接按下目标键；界面显示 `A`、`左 Alt` 等可读名称，后台自动保存对应 VK，并为常见扩展键自动勾选。随后选择按下/释放并插入或保存。
- 复制、删除、上移或下移事件。
- 按序号、类型或详情搜索，并循环定位上一个/下一个结果。
- 撤销或重做最近 20 步编辑。

事件时间线与兼容性报告之间的细分隔条可以上下拖动，以调整两个区域的显示高度。时间线上方提供插入、编辑、复制、删除、移动、搜索和撤销/重做；批量延时缩放和选中延时直接应用暂不显示。固定延时、键盘和鼠标事件仍可通过右键或双击打开编辑器。

若时间线出现“嵌套宏”但报告提示找不到目标：先把目标宏文件导入工作区，选中该“嵌套宏”事件，在工具栏“嵌套宏”下拉框选择目标，然后点击“绑定宏”。诊断会立即刷新。若要像雷云 4 一样主动加入一个子宏，选择目标宏和插入位置后点击“插入宏”。当前宏不会出现在自己的目标列表中；形成间接循环时会阻止导出并显示引用链。

主窗口可以在最大化、默认尺寸和紧凑尺寸之间自由调整；时间线会随窗口宽度重新排布，不需要使用横向滚动条。

编辑器不会自动补充按键释放。若宏结束时仍有按键或鼠标按钮处于按下状态，兼容性报告会显示错误并禁止导出。

## 4. 导出

选择目标格式：

- “雷云 4 宏 XML”：生成可单独导入雷云的 XML。
- “X-Mouse 宏文本”：生成可复制到 XMBC“模拟按键”的文本文件。

点击“安全导出”并选择新文件名。软件默认不覆盖原输入；覆盖已有输出时由 Windows 保存对话框再次确认。输出先写入同目录临时文件，完成后再替换目标。

## 5. 宏库

点击顶部“宏库”进入本地资料库。左侧选择“全部宏、收藏、最近使用、未分组、自定义分组或回收站”，中间管理 XMouse 文本，右侧管理雷蛇 XML。

- “新建”逐条填写名称、分组和 XMouse 宏文本；“导入”会复制外部 `.txt/.xml`，不会移动原文件。
- 双击或点击“在转换区打开”可进入现有时间线编辑器；修改后点击转换页“保存到宏库”，明确选择更新原条目或另存新条目。
- 在 XMouse 或雷蛇列表中直接拖动宏，可调整同栏显示顺序；绿色线表示插入位置，顺序会在刷新和重启后保留。搜索、回收站或“按时间排序”启用时暂不接受手动拖拽。
- 分组对应真实一级文件夹，根目录文件显示为“未分组”。PNG 等附件不会出现在宏列表，但重命名整个分组时会随文件夹移动。
- 删除进入宏库回收站，可恢复或永久清空；同名导入和恢复会自动使用 `(2)` 编号，重命名与移动不会静默覆盖。
- 在“设置”中选择宏库路径。切换路径不会搬动旧资料；“迁移宏库”只接受空目录，复制并校验完成后才切换。

## 6. 诊断说明

- 错误：会阻止导出，必须修复或改选目标格式。
- 警告：转换可以继续，但目标行为可能降级或需要确认。
- 信息：说明规范化、取消或其他非阻断状态。

未知事件不会被静默删除。如果目标格式无法安全表达某个事件，软件会保留可见诊断并停止导出。

## 7. 安全建议

- 始终保留雷云和 XMBC 原始宏备份。
- 首次转换后先在无风险应用中测试，不要直接用于不可逆操作。
- 不要运行来源不可信、包含程序启动或系统消息命令的宏。
- 软件日志不记录完整宏内容，但宏文件本身可能包含隐私或操作策略，请谨慎分享。

## 8. 卸载

退出软件后删除解压目录即可。软件不会安装系统服务或写入雷云/XMBC 配置。若已注册开发环境的日报定时任务，它不属于绿色发布包。
