## 风险

- `.synapse4` 的 `hash` 算法尚未确认，因此第一版只读不写整包。
- 雷云和 XMBC 存在未覆盖事件类型，必须通过匿名化样本逐步扩充。
- BOM 编码矩阵已覆盖外层输入，但其他本地代码页和无 BOM UTF-16 明确不自动猜测。
- Version 4 雷云 XML 已明确采用 VK 与零基 `0/1`；仅缺少 Version 且只含鼠标代码 `1` 的旧外部文件仍存在右键/兼容左键歧义，当前按旧兼容左键处理。
- `Type=6/LoopEvent` 循环结构已依据用户截图实现并自动回归，仍需用户用原宏确认展开后的实际播放次数与节奏。
- 扩展 XMBC 黄金样本已通过自动化语义往返，但尚未逐项验证实际执行效果。
- 用户宏可能包含隐私或游戏操作策略，测试样本必须最小化和匿名化。
- 三主题实际表面、按钮状态和可访问名称检查已自动覆盖，但多 DPI、完整键盘顺序和屏幕阅读器仍需人工验收。
- PerMonitorV2、正式发布 EXE 的嵌入 Manifest、紧凑布局、时间线虚拟化和结构化键盘/UIA 合同已自动验证；真实跨屏 DPI 切换和 Narrator 语音仍需补充人工验收。
